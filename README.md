## Getting Started

Create a `.env` file in the project root directory.

```bash
API_KEY=
RETURN_URL=
REALM=
PORT=
PRIVATE_KEY=
ADMIN_ROLE=
ADMIN_STEAM_ID=
```

```bash
# Install dependencies
$ npm install

# Start the server
$ npm start
```
